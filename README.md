# JPEG Documentation

Documentation for JPEG - A Social Game Integrating Photo Sharing, Prediction Markets, and User-Generated Content.

## Overview

JPEG is a pioneering social game that transforms photo sharing into a dynamic prediction market, blending BeReal's spontaneous creativity with robust financial incentives. This documentation provides comprehensive guides, technical documentation, and whitepaper content.

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview your documentation changes locally. To install, use the following command:

```bash
npm i -g mint
```

Run the following command at the root of your documentation, where your `docs.json` is located:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing Changes

Install the Mintlify GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) to propagate changes from your repo to your deployment. Changes are deployed to production automatically after pushing to the default branch.

## Documentation Structure

- **Getting Started**: Introduction, quickstart, and overview
- **Core Concepts**: How it works, user types, prediction market, tokenomics
- **Platform Features**: Themed photos, freestyle JPEGs, collects, royalties
- **Technical**: Mathematical modeling, behavioral economics, game theory, risks
- **B2B**: Business opportunities, case studies, data annotation
- **Development**: Stages, future plans, ethical considerations
- **Whitepaper**: Complete technical documentation
